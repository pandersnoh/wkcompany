# WK COMPANY
문화 마케팅 기업 WK COMPANY 웹사이트입니다.<br/>
웹기획자로 근무할 당시 직접 기획했던 사이트로<br/>
웹퍼블리셔로 업종 변경 후 학습 목적으로 제작하였습니다.<br/>
PC와 모바일로 분리되었던 기존의 서비스와는 달리 반응형 웹으로 재구성하였습니다.

구현결과 : https://laughing-murdock-9d6a32.netlify.app/
